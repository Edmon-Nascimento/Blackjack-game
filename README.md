# Blackjack Game

Interactive Blackjack game built with vanilla JavaScript, HTML and CSS.  
This project was created to practice core JavaScript fundamentals, DOM manipulation and game logic without using frameworks or libraries.

## Technologies

- HTML5
- CSS3

## Features

- Random card generation
- Basic Blackjack game rules
- Player betting system
- Dealer hand simulation
- Win / lose conditions
- Restart game functionality
- Dynamic DOM updates

---

## Objective

The goal of this project was to strengthen JavaScript fundamentals by building a complete interactive application without frameworks.

Main concepts practiced:

- DOM manipulation
- Functions
- Arrays
- Conditional logic
- Event handling
- State management
- Game flow control

---

## What I Learned

During this project I practiced and improved:

- JavaScript logic building
- Managing application state without frameworks
- Working with random numbers
- Updating the DOM dynamically
- Structuring game rules
- Separating responsibilities into functions
- Creating interactive UI with pure JavaScript
